Sysbadge User Manual

To start using your Sysbadge, follow these steps:

  1. Connect your badge display unit to your PC's USB port.
  2. On the back of the display, locate the {boot} and {rst} buttons.
  3. Press and hold {boot} for 3 seconds, then press {rst} while still holding {boot}.
  4. This will connect the display as a USB stick to your PC for file transfer.
  5. Download the firmware UF2 file from this link and copy it to your display.
  6. The badge will automatically disconnect from the PC. Reconnect it by following the same steps as above.
  7. Visit PluralKit to obtain your system ID.
  8. Enter your system ID on this page to get the data file, and copy it to your badge display.

After completing these steps, you can disconnect any power sources if connected.

Note: A power source is required for using your Sysbadge.

When powered on, the badge should display your system name. Press B for software and support information. Press B again to return to your system name, and use C to access the Name list. To edit a name, select the cell by pressing B, which will highlight it, and then use the up or down buttons to edit. You can display multiple people by pressing C again to show 2, 3, or 4 cells, and select the cell to edit with B.

If there's no activity for 30 seconds (configurable during build), the badge will enter sleep mode to save energy. It will return to the starting screen when it wakes up, as it can't save its state.

Optional:

For added protection, you can find a 3D printable case here. Assemble it using 2.5x 10mm screws or glue.

Compatible power sources include power banks and USB-C to USB-C cables.
